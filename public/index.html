Fedifinder

Extract the fediverse handles of your Twitter followings or list members and import them on Mastodon to follow them all at once.

This is the old version of fedifinder. Please try the new version at Fedifinder or Fedifinder Backup for a better experience.

Find fediverse handles of Twitter contacts

Fedifinder looks for @user@host.tld, user@host.tld and host.tld/@user in the name, description, location, url field and pinned Tweet.

Made by @Luca@vis.social. Source code in the fish menu (top right). If something doesn't work, please post an issue on Github or discuss the tool in the fediverse.

Other options

  • debirdify by @pruvisto@graz.social works very similarly.
  • twitodon needs you and your followers to be signed up there to match you.
  • Fedifinder Backup when Fedifinder is down.
